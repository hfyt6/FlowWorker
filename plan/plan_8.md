## 技术要点

### Token 估算公式
```csharp
// 近似估算（实际使用 Tiktoken）
tokens ≈ (characters / 4) + (words / 2)

// 消息 Token 计算
message_tokens = system_tokens + 
                 sum(message.content_tokens + message.role_tokens)
```

### 错误响应格式
```json
{
  "error": {
    "code": "API_ERROR",
    "message": "API 调用失败",
    "details": "连接超时",
    "suggestion": "请检查网络连接或稍后重试",
    "retryable": true
  }
}
```
